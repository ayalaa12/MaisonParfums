MAISON PARFUMS — VERSIÓN CORREGIDA

PARA ABRIR LA PÁGINA SIN COMPLICACIONES:

1. Descomprime completamente Maison-Parfums-CORREGIDO.zip.
2. Abre el archivo index.html que aparece junto a este documento.
3. Ese index.html es autónomo:
   - Incluye estilos.
   - Incluye JavaScript.
   - Incluye los 210 productos.
   - Incluye todas las imágenes actuales.
   - Funciona aunque esté separado de las demás carpetas.

PARA SEGUIR EDITANDO LAS IMÁGENES:

Abre la carpeta:

maison-parfums-editable/

Dentro de esa carpeta se conserva la versión profesional editable:

- index.html
- css/
- js/
- data/
- assets/

En esa versión, cambiar una imagen en:

assets/images/catalogo/<marca>/<Perfume>.webp

actualiza el perfume en toda la página.

IMPORTANTE:

No abras el index.html editable mientras todavía esté dentro del ZIP.
Primero usa “Extraer todo”.

DIFERENCIA ENTRE LAS DOS VERSIONES:

- index.html de la carpeta principal:
  Versión autónoma para visualizar y presentar. No depende de carpetas.

- maison-parfums-editable/index.html:
  Versión para continuar trabajando y reemplazar imágenes una sola vez.

No se eliminaron productos ni funciones.
