# Sistema central de imágenes

Cada perfume utiliza un solo archivo compartido:

`assets/images/catalogo/<marca>/<Nombre del perfume>.webp`

Ejemplo:

`assets/images/catalogo/lattafa/Khamrah.webp`

Esa misma imagen aparece automáticamente en Inicio, catálogo, buscador, marca, colecciones, quiz, wishlist, carrito, ficha y relacionados.

## Para cambiar una foto

1. Prepara la nueva imagen en WebP, de preferencia 1200 × 1500 px.
2. Ponle exactamente el mismo nombre del archivo existente.
3. Reemplázala dentro de `assets/images/catalogo/`.
4. Presiona `Ctrl + F5` en el navegador.

No modifiques HTML, JavaScript ni JSON.

Cada fragancia unisex aparece en una sola ficha y utiliza una sola imagen centralizada.
